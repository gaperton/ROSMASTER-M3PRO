## 2. Dify

### 2.1 Introduction

Dify is an open-source development and deployment platform for **production-grade agentic AI solutions**. Developers can quickly build, deploy, and scale AI applications without complex technical work.

![Picture: page 0: picture 11](_page_0_Picture_11.jpeg)

### 2.2 Features

#### Visual AI Application Building

Dify lets you design AI workflows and applications through intuitive drag-and-drop operations. It supports complex logic combinations and nesting, reducing the need for deep backend development. This makes it possible to move quickly from an idea to a working AI agent application.

#### Seamless Integration with Large AI Models

After applying for a model provider's API key, you can connect to hundreds of online model services.

#### Multi-Source Data Support

Dify can extract and transform data from multiple sources, automatically build indexes, and store them in a vector database. This provides efficient RAG (Retrieval-Augmented Generation) for large models and supports local RAG knowledge bases.

#### Rich Plugin System

Through Dify's plugin ecosystem, large AI models can be combined with tools and RAG knowledge bases to create AI agents for many vertical application fields.
